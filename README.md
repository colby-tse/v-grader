# V-Grader
A simple bouldering v-grade calculator for personal use.

## How it works
Give each category of the route a rating from 0 to 10. A simple average is then calculated. A route is considered "soft" if $0<x<0.5$, and "hard" if $x>=0.5$, where $x$ is the decimal part of the average.
